# 🚌 Where Is My Bus? – Real-Time Bus Tracking System for Small Cities #SafwanShaikh29

## 📌 Overview
**Where Is My Bus?** is a real-time bus tracking and monitoring system designed specifically for small cities and towns. The platform enables passengers to track bus locations live, view estimated arrival times, and access route information through an easy-to-use web interface.

## 🚀 Features
- 📍 Real-time bus location tracking
- 🗺️ Interactive route visualization
- ⏱️ ETA prediction
- 🚌 Bus stop information and schedules
- 🔄 Live updates
- 📊 Route monitoring dashboard
- 📱 Responsive web interface

## 🛠️ Technology Stack
### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python (Flask)
- Node.js

### Database
- CSV-based route and stop management
